# Client Reviews CPT

A WordPress plugin that registers a custom post type (CPT) for Client Reviews and provides a styled frontend slider using the Slick Carousel library via the `[client_reviews_slider]` shortcode.

## Features

- Custom post type for managing client reviews
- Responsive Slick slider with desktop and mobile layouts
- Easy to use shortcode: `[client_reviews_slider]`
- Supports titles, content, and thumbnails for reviews

## Installation

1. Download the plugin files.
2. Upload the plugin folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage

1. After activation, go to the WordPress admin dashboard.
2. Navigate to "Client Reviews" to add new reviews.
3. Add reviews with title (author name), content (review text), and featured image (author photo).
4. Place the shortcode `[client_reviews_slider]` on any page or post to display the reviews slider.
